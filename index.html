聊聊GE公司

It is a good time, it is a bad time.
国庆被GE刷屏了，Jeff提前退休（非自愿吧），@Beth Comstock 被要求年底离开，还有，还有。。。。。。
这不得不让我发布一篇之前聊聊GE公司的文字，只代表个人观点，不喜尽管喷。
加入GE已经快第七个念头了，我有话要说。
如果你认为GE能给你一份钱多，活少，离家近的工作的话，那你就大错特错了。
首先GE提供的福利待遇在市场上的并不具有竞争优势，有些岗位会比市场价格大打折扣。比如说占领CTP半边大楼的GOF团队中负责日常会计的小伙伴，每个月领到的薪水也只能算是糊口。其次，每个季度末的order，sales，convertible会让你喘不过气来。GE是一个不讲理的公司（哪个美国上市公司讲理）。因为GE是美国上市公司，每个季度各大投行的分析师会拍着脑袋给每个公司一个EPS的预估值。你一旦做不好，股票就会大幅度下跌。就算你做得好，那些拍脑袋的分析师会改拍大腿，然后下个季度会大幅度上调预期，直到把你搞死。
GE，衡量财务数据的KPI很简单，很粗暴。订单，季度转化订单，销售额，净利润和坏账。每过一个季度，就如同死了一会的感觉。

你认为进入GE就能学到高级的管理经验，与众不同的技能的话，我劝你还是不要进来的好。GE是一个熔炉，关键问题你是什么。如果你是块氧化铁，碰巧遇到个有学识、有 眼界的小领导，你会学到很多同时少走弯路。我听说很多GOF的小姑娘，或者FMP都是国外留学归国的有识之士，英语好、分析能力强、三维也不错，不过开始干的也都是搜集，整理数据的初级工作，GOF的财务工作即时给你五年时间也只能看到利润表的几个科目而已。

John rice极力打造的GOF团队（share service）占据了张江GE总部的大半写字楼，而如今随着他老人家的退休也将变成历史。shared service其实是把双刃剑，做好很难。

我觉得GE现在越来越没落。从不推崇6 sigma，到取消EMS改为PD的给别人点赞或写insight。从看得见摸得着的批评与自我批评改为无关痛痒的相互寒暄，甚至吹捧。人的本质喜欢听好话，谁喜欢忠言逆耳那？

一句话，GE缺钱。要不然也不会把可以卖的都卖掉。

GE，我之所以爱过你，是因为你的伟大，你的包容性与你那么多的优点。虽然你一直强调不变是变化，而你现在变得让人看不清，摸不透，看不到希望。你变得过于保守，变得胆小怕事，变得跟不上市场了。 也许，几年后瘦死的骆驼还是比马大。
吐槽归吐槽，明天还是要上班的。

New idea 有话说

If you want to rent a house, use Airbnb;
If you want to rent a car, use Uber;
If you want to rent a brain, use us, please.
开篇不谈“分享经济”，便称“大师”也枉然。
共享经济，是眼下最炙手可热的名词。其本质是弱化“拥有权”，强调“使用权”。无论是资源还是技能，对于拥有者而言因为私有化而没有得到充分利用，如今得到更广阔的价值。主要特点是，一个由第三方创建的、以信息技术为基础的市场平台。个体借助平台，交换闲置物品，分享自己的知识、经验，或者向企业、某个创新项目筹集资金等。共享经济的五个要素分别是：闲置资源、使用权、连接、信息、流动性。
有话说做的是知识与经验的共享平台。一种人的知识、智慧、经验、技能通过互联网转换成实际收益，从而达到各取所需的互联网新模式。试想当你职业发展遇到瓶颈的时候，或当你有个很好的想法希望付诸实现的时候，亦或当你的家庭生活遇到问题的时候，你是否想立刻找个“对”的人咨询一下？
与《知乎》，《在行》不同，有话说要做的是低门槛的分享平台。我们认为每一个人都有让别人钦佩的本领，有时候却没有发觉。我们发现它，培养它，让它生根发芽最后长成参天大树。有些人的汲汲渴求，是其他人的力所能及。分享的本质在于知识与经验。我们崇尚终身学习，成长是一辈子的事情。随着社会，经济以及家庭生活压力的增大，人们会觉得实现自我理想越来越难。学习局限于书本，工作和生活积累。反而忘记了分享行业知识和经验教训来得更直接，更解决问题。
分享，既满足他人需求，又实现自我价值的提升，同时将无形的知识转化为金钱。
我做过一个市场调查，发现大多数职场人士都有心理咨询方面的需求，有职场晋升和职业发展方面的需求，需要在孩子早期教育上获得指导与帮助。这些在现如今的中国并不得到足够的重视。人们并没有意识到心理的疾病，孩子的教育问题以及职业瓶颈是新的三座大山。它们没有被重视，反而越发野蛮直到悲剧发生。
同时，在中国不缺少提供这种知识和经验分享的人群。我们将供给端大致分为两类分享者。一类是在本职工作上有建树、有经验、有故事的他们，非常愿意把干货分享给其它人。他们大多是高等学历毕业，有留学背景，活动于北上广深，从事金融，娱乐，融资等行业。另一类是自由职业者。他们并没有朝九晚五的工作限制，但他们凭手艺吃饭。他们希望有人可以把能力变现，既帮助别人也满足了自己的需求。他们有固定的圈子，也存在需求没有被满足的时候。
知乎提供的单一的问题解答，然而很多事情并不是一个问题可以解决的。在行提供的针对性服务，但是这种1对1的服务效率低，有悖于分享经济的本质，同时行家的知识并不能得到广泛的，无时间、空间限制的传播。有话说使用视频专访，音频以及文字的形式将行家的思想、经验、亦或技能通过APP（小程序）平台分享给有需要的学员，从而让学习更有效。可以根据学员的需求，针对性的解答。
平台的收入来自：

  1. 学员的点播和会员费用10%
  2. 打赏的10%–学员可以通过打赏提问题，并得到解答
  3. Forum—直播各种forum，在线收取点播费

我们坚信成长是一辈子的事情，我们愿意通过互联网的方式让更多的人分享知识，同时点亮他人的人生。
